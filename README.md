# 个人学习网站

个人学习记录 · 实用工具集合 · 技术心得分享

## 项目简介

这是一个基于 GitHub Pages 搭建的个人学习网站，包含多个实用的在线工具和技术心得分享。

## 功能特性

### 📝 爬取资源心得
- **浏览器开发者工具实战**：详细讲解如何使用浏览器自带的开发者工具分析并获取网页资源

### 🎯 选择困难工具
- **选择困难症转盘**：自定义选项、设置概率权重，随机抽取结果
- **抛硬币**：一键在线抛硬币，随机生成正反面结果
- **快速选择**：输入选项，立即获得随机选择结果

### 🔧 实用工具
- **音频格式转换器**：支持各种音频格式转换为 MP3 或 WAV
- **进制转换器**：支持二进制、八进制、十进制、十六进制互转

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Audio API
- LameJS (MP3编码)

## 项目结构

```
├── index.html                  # 主页
├── audio_converter/           # 音频转换器
├── base_converter/            # 进制转换器
├── crawler_experience/        # 爬取心得
├── decision_wheel/            # 选择困难工具
├── common/                    # 公共资源
│   ├── css/                   # 公共样式
│   └── js/                    # 公共脚本
└── favicon_io/                # 网站图标
```

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可访问
2. 或通过 GitHub Pages 部署后访问

## 部署

本项目已配置为 GitHub Pages，可直接在 GitHub 仓库设置中启用 Pages 功能。
