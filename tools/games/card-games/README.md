# 桌游合集（card-games）

## 目录说明

本目录采用**历史兼容**的多 HTML 入口，以保持外链与搜索引擎已收录的 URL 不变：

- 合集入口：`index.html`（路径 `/tools/games/card-games/`）
- 各游戏单页：`texas.html`、`uno.html`、`hearts.html`（路径分别为 `/tools/games/card-games/texas.html` 等）

共享样式与脚本放在 **`assets/`**（含 `game-layout.css`、各游戏逻辑与 `assets/common/` 下的共用模块）。

全站目录约定与例外说明见仓库根下 **`docs/README.md`** 中的「前端目录约定」一节。
